# You & Me Diary 设计决策记录

更新时间：2026-04-26

## 1. 产品定位

`You & Me Diary` 是一个面向准妈妈的私密 AI 陪伴日记。

它不是孕期知识工具、母婴社区或医疗诊断产品，而是一个围绕 `you & me` 感受建立的个人空间：妈妈通过文字、语音、照片记录当天状态，AI 将这些碎片整理成可回看的日记图、温柔安慰、时间线和纪念册。

核心价值：

- 帮妈妈理解今天的自己
- 帮妈妈在疲惫、焦虑、身体不适时获得一点自我安慰
- 帮妈妈和宝宝建立更强的连接感
- 把孕期片段整理成未来可以送给自己和宝宝的礼物

## 2. 产品气质

关键词：

- 私密
- 温柔
- 简洁
- 有礼物感
- 像日记，不像管理工具

避免方向：

- 过度母婴电商感
- 重社交
- 医疗诊断语气
- 过多解释说明
- 仪表盘式信息堆叠

## 3. 页面结构

当前最终页面结构：

```text
Home
  ├─ Record Today
  │   └─ Generating
  │       └─ Result
  ├─ Timeline
  │   └─ Result by selected day
  ├─ Memory Book
  │   └─ Result by favorited page
  └─ Settings
```

## 4. Home 首页

首页只承担入口职责，像一本日记的封面。

保留内容：

- 产品名
- 设置入口
- 简短欢迎语
- 最近一句温柔提示
- `记录今天`
- `打开时间线`
- `打开纪念册`

已决定移除：

- 首页直接展示配色方案
- 过长产品介绍
- 结果卡片预览

## 5. Settings 设置页

设置入口位于首页右上角。

当前包含：

- 用户名
- 预产期
- 配色方案

配色方案：

- 奶油雾粉
- 鼠尾草
- 晨雾蓝
- 落日晚杏

用户名会影响首页称呼。预产期暂时仅作为用户资料保留，后续可用于孕周计算和提示语个性化。

## 6. Record Today 记录页

记录页采用类似 AI agent 输入框的方式，而不是表单。

输入形式：

- 文本输入
- 语音按钮
- `+` 附件按钮
- 本地图片附件
- `提交`

已决定移除：

- 手动选择情绪 tag
- 拆成语音、图片、情绪三段式输入

原因：

用户只需要表达当天发生了什么，情绪、身体状态、重要事件应由模型自动提取。

## 7. Generating 生成中页

生成中页面保持轻量，仅作为从输入到结果的情绪过渡。

定位：

- 给用户一点等待反馈
- 保持温柔气质
- 不展示复杂进度或技术细节

## 8. Result 结果页

结果页是当前产品最核心的页面。

最终结构：

```text
Result
  ├─ 状态：已自动记入时间线
  ├─ 日记图片册
  │   ├─ 左右滑动图片
  │   ├─ 图片上的注释按钮
  │   ├─ 收藏 icon
  │   └─ 分享 icon
  ├─ 给你的小安慰
  ├─ 注释详情
  │   ├─ 标题：例如“恢复感”
  │   ├─ 语气切换：我 / 宝
  │   ├─ 默认只读文本
  │   └─ 点击笔进入编辑，再点返回退出编辑
  └─ 长图预览
      ├─ 当前图片
      ├─ 展开的解释文本
      └─ 下载长图
```

### 8.1 日记图片册

Result 的主体验不是文本总结，而是一组由用户图片和 AI 文案生成的日记图。

每张图包含：

- 用户上传图的视觉气质
- 颜色提取或模拟色板
- AI 提炼出的主句
- 图片注释点

用户可以：

- 左右滑动查看不同图片
- 点击图片上的注释按钮
- 收藏当前图页
- 生成分享长图

### 8.2 注释详情

注释详情不再使用 `给自己 / 给宝宝 / 分享卡` 三个大 tab。

当前决策：

- 标题只显示当前注释标签，例如 `恢复感`
- 右侧用小 icon 切换语气
- `我` 表示写给自己
- `宝` 表示写给宝宝
- 文本默认只读
- 点击笔进入编辑
- 编辑状态下按钮变成返回
- 返回后文本恢复只读展示

### 8.3 给你的小安慰

图册下面先展示 `给你的小安慰`。

它是当日整体状态的温柔回应，不参与 tab 切换。

语气要求：

- 先理解，再安慰
- 不说教
- 不像医疗建议
- 给用户“今天可以不用那么强”的许可感

### 8.4 收藏与分享

收藏和分享都作为图册内的 icon 行为。

收藏：

- 当前图页加入 `Memory Book`
- `Timeline` 默认记录所有天
- `Memory Book` 只收主动收藏的图页

分享：

- 点击斜上箭头生成当前图页的长图预览
- 长图包含图片和展开后的解释文本
- 用户下载后可以自行裁剪
- 目标场景是朋友圈或小红书

## 9. Timeline 时间线

Timeline 采用日历式视图。

结构：

- 月视图
- 每天一个小圆点
- 圆点上显示 emoji
- 圆点颜色表示当天情绪或状态
- 点击某一天进入当天 Result

定位：

- 记录所有日常
- 不做精选
- 适合快速回看孕期状态变化

与 Memory Book 的区别：

- Timeline 是完整路径
- Memory Book 是主动收藏的纪念页

## 10. Memory Book 纪念册

Memory Book 只收用户主动收藏的图页。

定位：

- 更像未来礼物
- 不保存所有记录
- 只保存值得留下来的页面

每个收藏项包含：

- 图页缩略图
- 日期
- 图页标题
- 简短描述
- 重新打开入口

## 11. 视觉与字体

中文字体栈：

```css
"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Noto Sans SC", "Source Han Sans SC", sans-serif
```

设计方向：

- 简洁
- 轻柔
- 卡片感适中
- 尽量减少说明文字
- 用 icon 和状态表达交互

## 12. 安全边界

产品不做医疗诊断。

涉及腰疼、身体不适、胎动变化等内容时，只做：

- 记录
- 情绪理解
- 就医前问题整理
- 异常情况提醒咨询医生

避免输出：

- 明确诊断
- 治疗方案
- 药物建议
- 替代医生判断
